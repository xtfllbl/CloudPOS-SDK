# Contactless Card Distance

### Overview

This section outlines the effective reading distances for contactless cards when used with different models of Smart POS terminals.

### Reading Distance Specifications

* Model Q1:
  * Effective reading distance: 0 to 2 cm.
* Model Q2:
  * Effective reading distance: 0 to 4 cm.
* Model Q3:
  * Effective reading distance: 0 to 4 cm.



{% hint style="info" %}
**Note:**

* These distances represent the maximum range at which the POS terminal can successfully read a contactless card.
* Users should ensure that the card is within this range for optimal functionality and transaction processing.
{% endhint %}
