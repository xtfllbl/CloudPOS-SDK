# Resolve Serial Timeout

### Description

This issue occurs when there is a timeout during the serial reception of a key in the POS system.

Shows like the follow picture:

![](<../../.gitbook/assets/image (59).png>)



### Possible Causes

1. Date and Time Mismatch: The date and time settings between the key delivery POS and other terminals may not be synchronized.
2. Cable Connection Issues: There could be a problem with the connecting cable, such as incorrect connection or damage.
3. Incorrect Delivery Type Selection: The delivery type chosen for the key transfer might not be appropriate for the operation.
