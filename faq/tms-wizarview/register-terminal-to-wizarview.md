# Register Terminal to WizarView

1. Administrator Login:
   * Start by logging into the terminal as an Administrator. This step is essential to access the required settings for registration.
2. Accessing Settings:
   * On the terminal's main screen, navigate to 'Settings'. This is typically found in the main menu or as an icon on the home screen.
3. Go to About POS:
   * In the Settings menu, select 'About POS'. This section usually contains information about the POS system and related configurations.
4. Navigate to POS Configuration:
   * Within the 'About POS' section, find and select 'POS Configuration'. This will lead you to more specific settings related to the POS system.
5. Select Remote Management:
   * In the POS Configuration options, choose 'Remote Management'. This is where you can manage settings for remote services and applications.
6. Register with WizarView:
   * In the Remote Management menu, look for the option to 'Register WizarView'. Select this to initiate the registration process.
7. Check the Notice Bar:
   * After attempting registration, check the notice bar on your terminal. It will display a message indicating whether the registration was successful.

Note: It's important to ensure that you have a stable internet connection during this process to avoid any interruptions or errors.
