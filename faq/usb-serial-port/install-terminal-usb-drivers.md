**Issue: Fastboot Not Detecting Device in Windows 10**

* **Problem Description:**
* Sometimes, in Windows 10, Fastboot may fail to detect the device. This issue typically indicates that the USB driver for Fastboot needs to be reinstalled.

1. Download the Fastboot USB driver [here](https://ftp.wizarpos.com/advanceSDK/usbdriver_w10_fastboot.zip).
2. Unzip the downloaded file.
3. Right-click on the file named 'android\_winusb.inf'.
4. Select 'Install' from the context menu.
5. Once the installation is complete, Fastboot should function correctly.
