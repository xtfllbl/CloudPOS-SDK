# Install Browsers

### Overview

Some Q-Series models may have outdated browsers or lack a pre-installed browser. If your application requires a browser, you can manually install a browser APK on the terminal.

### Recommended Browser for Q2

* **Q2 Browser:** Specifically designed for Q2 devices, this browser can be obtained and installed for optimal compatibility. [Download Q2 browser](https://ftp.wizarpos.com/advanceSDK/Browser-q1_platform.apk)

### Alternative Browsers

* **Other Browser APKs:** You are also free to install any other compatible browser APKs on your Q-Series device as needed.
