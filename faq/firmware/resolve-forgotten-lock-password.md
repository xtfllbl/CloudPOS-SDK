# Resolve Forgotten Lock Password

To address the issue of a forgotten security lock password, the recommended solution is to perform a factory reset.

#### Steps to Perform a Factory Reset

1. **Reference Guide:** [Detailed instructions on performing a factory reset](../settings-menu/perform-factory-data-reset.md).
2. **Caution:** Be aware that a factory reset will erase all data on the device. Ensure that important data is backed up before proceeding.
